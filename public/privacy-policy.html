1. Données collectées

Ce site web collecte uniquement des données techniques nécessaires à son fonctionnement et à l'analyse du trafic :

  • Données de navigation : Pages visitées, temps passé sur le site, source du trafic
  • Données techniques : Type de navigateur, appareil utilisé, adresse IP (anonymisée)
  • Préférences de consentement : Votre choix concernant les cookies analytics

2. Finalité de la collecte

Les données sont collectées pour :

  • Analyser le trafic et améliorer l'expérience utilisateur
  • Comprendre les préférences des visiteurs
  • Optimiser les performances du site

3. Utilisation des Cookies

Cookies Analytics (Google Analytics)

Ce site utilise Google Analytics pour analyser le trafic et améliorer l'expérience utilisateur. Les données collectées sont :

  • Pages visitées
  • Temps passé sur le site
  • Source du trafic
  • Informations techniques (navigateur, appareil)
  • Interactions avec les éléments du site (clics, téléchargements)

Important : Ces données sont anonymisées et ne permettent pas de vous identifier personnellement. Google Analytics n'est activé que si vous donnez votre consentement.

4. Base légale

La collecte de données est basée sur :

  • Consentement : Pour les cookies analytics (vous pouvez refuser)
  • Intérêt légitime : Pour les données techniques nécessaires au fonctionnement du site

5. Durée de conservation

  • Données analytics : 26 mois maximum (Google Analytics)
  • Préférences de consentement : Jusqu'à ce que vous les modifiez
  • Données techniques : Supprimées automatiquement à la fermeture de la session

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Demander quelles données sont collectées
  • Droit de rectification : Corriger des données inexactes
  • Droit à l'effacement : Demander la suppression de vos données
  • Droit d'opposition : Refuser le traitement de vos données
  • Droit à la portabilité : Récupérer vos données dans un format structuré
  • Droit de retirer le consentement : À tout moment pour les cookies

7. Gestion des Cookies

Vous pouvez à tout moment :

  • Accepter ou refuser les cookies via la bannière de consentement
  • Modifier vos préférences en supprimant le cookie de consentement
  • Désactiver les cookies via les paramètres de votre navigateur
  • Utiliser le mode navigation privée pour éviter le stockage de cookies

8. Sécurité

Vos données sont protégées par :

  • Le chiffrement HTTPS sur tout le site
  • L'anonymisation des données analytics
  • L'absence de stockage de données personnelles sensibles
